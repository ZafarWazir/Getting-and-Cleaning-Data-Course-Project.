# CodeBook

## Data Source
Data collected from the accelerometers of the Samsung Galaxy S smartphone.

## Transformations
1. Merged training and test sets.
2. Extracted mean and standard deviation measurements.
3. Used descriptive activity names.
4. Labeled data set with descriptive variable names.
5. Created tidy data set with the average of each variable for each activity and each subject.

## Variables
- subject: ID of the test subject.
- activity: Type of activity performed.
- All other variables: Numeric measurements from the smartphone sensors.
